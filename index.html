What's MateriApps LIVE!?

A number of high-performance and high-precision application software for computational materials science simulation have been developed in Japan. The experimentalists and researchers in the private companies, however, don't even know the existence of such applications, and are mainly using open-source software from abroad or commercial software. CMSI, Computational Materials Science Initiative, is developing MateriApps, a portal site of materials science simulation, aiming at introducing the features and characteristics of various materials science simulation programs that have not been known widely or that are being developed in years to come. As for the open-source software, we also provide an environment with fully-developed manuals and tutorials in order to make it easy for users to try out the software.

日本国内において、計算物質科学シミュレーションのための、高性能・高精度な手法・ソフトウェアが数多く開発されています。しかしながら、実験家・企業研究者にその存在が伝わっておらず、主として海外製の公開ソフトウェア・商用ソフトウェアが使われているのが現状です。計算物質科学イニシアティブ(CMSI)では、物質科学シミュレーションのポータルサイトMateriAppsの整備を通じ、これまでにあまり知られていなかった国内外で開発された物質科学アプリケーション、あるいは今後開発されるアプリケーションを紹介しています。特に、公開されているソフトウェアについては、充実したマニュアル・チュートリアルなど、利用者が直接ダウンロードして気軽に試すことのできる環境を提供しています。

In the meantime, when trying to try out open-source software, installation of software itself often becomes a serious obstacle. Although CMSI is currently advancing installation of typical applications to domestic supercomputer sysytems, not all people can have access to such environment.

一方で、公開ソフトウェアを試そうとしたとき、ソフトウェアのインストールそのものが大きな障害となっているのも事実です。CMSIでは、国内の代表的なスーパーコンピュータへの代表的なアプリケーションのインストールなども進めていますが、必ずしも全ての人々がそのような環境を利用できるわけではありません。

MateriApps LIVE! offers an environment where one can try out computational materials science simulation freely, using a notebook PC, etc. All environment required to begin tutorials, such as MateriApps applications, OS (Debian GNU/Linux), editors, and visualization tools, is provided in a USB memory stick.

MateriApps LIVE!は、手持ちのノートPCなどを用いて、気軽に計算物質科学アプリケーションを試せる環境を提供します。MateriAppsアプリケーション、OS (Debian GNU/Linux)、エディタ、可視化ツールなど、チュートリアルを始めるのに必要な環境は全て1本のUSBメモリに収められています。

How to get MateriApps LIVE!

  • Download hybrid ISO image file and write into a USB memory stick (How to make MateriApps LIVE! USB)
  • Download and import VirtualBox virtual hard disk image (OVA), and use it in a virtual environment (MateriApps LIVE! OVA version)
  • At CMSI application tutorials held by CMSI or exhibitions
  • ハイブリッドISOイメージファイル(OVA)をダウンロードし、USBに書き込む (How to make MateriApps LIVE! USB)
  • VirtualBox仮想ハードディスクイメージをダウンロード・インポートし、仮想環境で利用する (MateriApps LIVE! OVA版)
  • CMSIが主催/共催するアプリーケーション講習会・学会会場での配布

How to use MateriApps LIVE!

MateriApps LIVE! can be started in various ways:
  • Boot directly from MateriApps LIVE! USB
  • Boot from MateriApps LIVE! USB in a virtual machine, such as VirtualBox, VMWare Player, Parallels
  • Install MateriApps LIVE! to your PC from MateriApps LIVE! USB (CAUTION: installation process will overwrite HD in your PC and the whole contents on HD may lost completely.)
  • Install MateriApps LIVE! to a virtual machine from MateriApps LIVE! USB (document will be available soon)
  • Use MateriaApps LIVE! in a cloud environment, such as Amazon Web Services (will be available soon)
MateriApps LIVE!は様々な方法で利用可能です。
  • MateriApps LIVE! USBから直接ブート
  • 仮想マシン(VirtualBox, VMWare Player, Parallelsなど)の中でMateriApps LIVE!イメージからブート
  • MateriApps LIVE! USBからPCにMateriApps LIVE!をインストールして利用 (注意! 現在のハードディスクの内容は上書き消去されます)
  • MateriApps LIVE! USBから仮想マシンにMateriApps LIVE!をインストールして利用(ドキュメント準備中)
  • クラウド環境(Amazon Web Servicesなど)での利用(準備中)

How to boot from MateriApps LIVE! USB

  1. Plug in MateriApps LIVE! USB to your PC
  2. Switch on PC
  3. Continue pressing a (system dependent) specific key until a boot device menu opens.
    • In many PCs, explanation of the keys (F2, F12, etc.) for opening a boot device menu is displayed on a screen upon BIOS startup.
    • In Mac OS X, continue pressing ALT (OPTION) key.
    • Under investigation for Windows 8
  4. Choose the USB memori stick from the list of boot devices.
  5. Choose language and type of keyboad from boot menu of MateriApps LIVE!

For more detailed information, please look at MateriApps LIVE! / Getting Started.

  1. PCの電源を切った状態でMateriApps LIVE! USBを差し込む
  2. PCの電源を投入
  3. ブートデバイスメニューが開くまで、特定のキー(システム依存)を押し続ける
    • 多くのPCでは、BIOS起動時に画面にブートデバイスメニューを開くためのキー(F2, F12など)の説明が表示される
    • Mac OS Xでは、ALT キーを押し続ける
    • Windows 8については調査中
  4. ブートデバイスの一覧からUSBメモリを選択し起動
  5. MateriApps LIVE!の起動メニューから、言語・キーボードを選択し、起動

詳しくは、「MateriApps LIVE! / はじめの一歩」をご覧ください
